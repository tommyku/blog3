Physical Web is about the web

Physical web is QR code without scanning.

After a lengthy discussion with friend this is what we came up with if we had to describe the Physical Web in one sentence.

The core technology behind Physical Web is becons working with Eddystone beacon format based on Bluetooth.

A Beacon, a little device to place just anywhere, continuously broardcasts a URL via Bluetooth to the surrounding area. Any device nearby can see an URL broadcasted by the beacon and those who open the URL from a browser can interact with either an online service or the device connected to the beacon itself (via Web Bluetooth).

So yes, technically you can use the Physical Web to connect to your trash can and turn it into an R2-D2 from any device that has Bluetooth, browser, and can recognize the Eddystone format.
